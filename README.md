# 一米菜园设计器

代码来自 https://github.com/phillipdupuis/garden-planner 因为里面带着后端代码看不懂，把前端代码独立出来，并增加 ios 应用，方便自己使用

## 配色方案

- 当然，明亮的、鲜艳的、有趣的配色是很多卡通颜色的特点。以下是一个建议的配色方案：

1. 主题颜色：藏青色 (#0066CC)
2. 进行强调的颜色：闪光蓝色 (#3399FF)
3. 辅助色 1：荧光绿 (#99CC00)
4. 辅助色 2：阳光黄色 (#FFCC00)
5. 辅助色 3：活力粉色 (#FF3399)
6. 背景色：白色 (#FFFFFF)
7. 基础文字色：墨石色 (#333333)
